"Propósito del sitio"


El propósito de este sitio es aprender mediante problemas de la vida real a comprender de una mejor manera temas como las etiquetas semánticas, CSS Grid


Con la realización de esta fase aprenderemos con ejemplo significativos como podemos usar estas herramientas para dar un mejor contorno a nuestra página web, paso a paso en la medida que avanzamos vemos una estructuración que es amena a los ojos del usuario que quiera utilizar nuestra página.
