Etiquetas Semánticas

En los nuevos estándares de la utilización del leguaje HTML5, para la creación de páginas más accesibles para todo el público, por esto ha incluido las etiquetas semánticas con el fin de hacer más sencillo el código a la hora de ser presentado, en decir que estás servirán para dividir un documento en partes lógicas, indicando el tipo de contenido que se incluye en cada parte, con estas se mejora la accesibilidad para toda clase de persona que se enfrenta con el código. EL HTML es un lenguaje de programación que utiliza un tipo de códigos llamados etiquetas, las cuales definen los elementos que componen la página web: texto imágenes, entre otras. A través de su historia estas etiquetas han evolucionado, convirtiéndose más funcionales apareciendo las etiquetas semánticas las cuales han ayudado a indicar que son y cuál es el significado de lo que contiene dentro. En los nuevos estándares de la utilización del leguaje HTML5, han incluido las etiquetas semánticas con el fin de hacer más sencillo el código a la hora de ser presentado, en decir que estás servirán para dividir un documento en partes lógicas, indicando el tipo de contenido que se incluye en cada parte, con estas se mejora la accesibilidad para toda clase de persona que se enfrenta con el código, esto facilita a que las páginas sean accesibles para todo el público.

Diez, T., Domínguez, M. J., Martínez, J. J., & Sáenz, J. (2012). Creación de páginas Web accesibles con HTML5. Consultado el, 22. Recuperado de //http://www.esvial.org/wp-content/files/Atica2012_pp120-129.pdf

CSS Grid

Como es conocido el CSS en un lenguaje que se integra con HTML5, en este caso el CSS Grid es un sistema de diseño web que no ayuda a dividir la pagina en una cuadricula, por lo cual nos ayuda a distribuir los elementos en una forma más sencilla. Como es conocido el CSS en un lenguaje que se integra con HTML5, en este caso el CSS Grid es un sistema de diseño web que no ayuda a dividir la pagina en una cuadricula, por lo cual nos ayuda a distribuir los elementos en una forma más sencilla. Sus múltiples ventajas, ayudan a que cualquier navegador soporte su código, esta ayuda que a facilitar el trabajo utilizando menos códigos, con lo que ayuda a optimizar tiempo y recursos, también nos ayuda a crear elementos más dinámicos que se adaptan a diferentes tamaños y resoluciones.

Atkins Jr, T., Etemad, EJ y Atanassov, R. (2013). Diseño de cuadrícula CSS. Recuperado de https://www.w3.org/TR/2013/WD-css3-grid-layout-20130402/
